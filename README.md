# Projeto-de-Circuitos-Eletr-nicos---IoT
Projeto em Tinkercad simula uma estufa com Arduino, integrando sensor de temperatura, motor, LED e buzina. O sistema lê a temperatura, aciona o ventilador a partir de 30 °C e, acima de 50 °C, ativa LED e alarme. Todo o controle é programado em linguagem C no Arduino.
